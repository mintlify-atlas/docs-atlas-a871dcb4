## Quick Start

Get up and running with JRE Notion Workers in minutes.

<Steps>
  <Step title="Install dependencies">
    Install the required packages using your preferred package manager.
    
    <CodeGroup>
    ```bash npm
    npm install
    ```
    ```bash bun
    bun install
    ```
    </CodeGroup>
  </Step>

  <Step title="Configure credentials">
    Set up your environment variables with Notion API credentials.
    
    ```bash
    cp .env.example .env
    # Edit .env with your NOTION_TOKEN and database IDs
    ```
    
    <Note>
    Required variables: `NOTION_TOKEN`, `DOCS_DATABASE_ID`, `HOME_DOCS_DATABASE_ID`, `TASKS_DATABASE_ID`
    </Note>
  </Step>

  <Step title="Register workers">
    The three workers are automatically registered when you run the project.
    
    ```bash
    bun run dev
    ```
    
    You can now invoke `write-agent-digest`, `check-upstream-status`, or `create-handoff-marker` from your agents.
  </Step>
</Steps>
